Task 1
Написать класс, который вычисляет сумму чисел от 0 до заданного числа.
В классе должно быть два метода: вычисление через цикл и рекурсивное вычисление.
В другом классе Application создать объект класса-калькулятора и вызвать оба метода.
Сравнить результат.

Task 2
Написать класс, который вычисляет произведение чисел от 0 до заданного числа, с шагом 2. Пример:
передаем число 15: 15х13х11х9х7х5х3х1
В классе должно быть два метода: вычисление через цикл и рекурсивное вычисление.
В другом классе Application создать объект класса-калькулятора и вызвать оба метода.
Сравнить результат.

Task 3
Написать класс для проверки, является ли слово палиндромом, с помощью рекурсии.
При написании программы можно использовать методы s.charAt() и s.substring()
