task 1
Работа с транзакциями. Через pgAdmin в одной транзакции создайте новый заказ (запись в таблице orders) и три
строчки заказа (записи в таблице order_items). Полученный скрипт скопируйте в текстовый файл в ваш проект.

task 2
Работа с транзакциями через JDBC. В джава-коде в одной транзакции создайте новый заказ (запись в таблице orders) и три
строчки заказа (записи в таблице order_items).

task 3
Создайте новый проект через https://start.spring.io/ по аналогии с проектом, который мы делали на занятии.
Импортируйте в IDEA, добавьте в зависимоти HikariCP.
Создайте в проекте класс UserRepository c методом List<User> findAllUsers(), класс UserService, который вызывает
UserRepository и возвращает список строк вида 'Sergey Ivaonv' - то есть имя+фамилия всех пользователей.
Создайте класс GlobalService, который имеет ссылку на UserService и в методе init() запрашивает имена пользователей и
выводит их в консоль.
