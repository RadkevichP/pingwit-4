Task 1
Создайте новый мавен-проект, через maven quickstart archetype.
Измените версию джавы на 17, добавьте в проект зависимости на
        <dependency>
            <groupId>org.junit.jupiter</groupId>
            <artifactId>junit-jupiter-engine</artifactId>
            <version>5.9.2</version>
            <scope>test</scope>
        </dependency>
        <dependency>
            <groupId>org.assertj</groupId>
            <artifactId>assertj-core</artifactId>
            <version>3.24.2</version>
            <scope>test</scope>
        </dependency>
в проекте создайте один класс GreetingService с одним методом greet, короты будет принимать на вход строку,
а возвращать строку с префиксом 'Приветсвую тебя, '. пример: greetingService.greet("Pavel") вернет "Приветсвую тебя, Павел"
Если на вход пришла строка, содержащая цифры, метод должен бросить IllegalArgumentException с сообщением 'имя не может содержать цифры!',
если пришла пустая строка или null - сообщение в эксепшене должно быть 'имя не может быть пустой строкой!'
Если есть необходимость- можете добавить в проект библиотеку Apache StringUtils
Наш проект - библиотека, поэтому в нем не должно быть класса с методом 'main'. проверьте работу метода, написав для него тесты, которые проверят
различные варианты аргументов.

Выполните clean, test, package, install.
Проверьте вручную, появилась ли ваша библиотека в директории м2 на вашем компьютере

После этого, скопируйте ваш класс и тест в проект с домашними заданиями и запушьте для проверки.
