Task 1
Создайте класс Motorcycle, в котором должны быть поля brand, model, engineVolume (объем двигателя в см3),
String condition.
В классе application создайте объект типа Motocycle и сериализуйте его с помощью java-сериализации. Поле
condition сериализоваться не должно!
Далее в этом же классе десериализуйте Motocycle из файла и выведите его в консоль

Task 2
В пакете task2 лежат 4 файла trackXXX.json. Скопируйте их себе в проект. Посмотрите на поля в файле
track1.json и напишите ваш класс, в который можно будет десериализовать этот объект.
В классе Application десериализуйте все 4 трека, содайте из них массив и распечатайте полученный массив.
Используйте для десериализации библиотеку jakson, пример смотрите в коде с занятия.
В этой версии решения не используйте циклы - 4 раза создавайте BufferedInputStream bis = new BufferedInputStream(new FileInputStream(...))
